# TelecomX1
Projeto de análise de evasão de clientes (churn) com aplicação completa do processo de ETL. Os dados são extraídos de uma API, tratados e transformados para análise exploratória. O trabalho inclui limpeza, organização e geração de insights estratégicos, além de relatório detalhado com conclusões que apoiam decisões e futuros modelos preditivos.
